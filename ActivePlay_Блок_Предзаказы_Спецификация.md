> Версия: 1.0 | 20 марта 2026
> Для: разработка главной страницы + будущий AI-агент

---

## 1. Что показываем

Карточки игр, которые **реально доступны для покупки по предзаказу** на PlayStation Store прямо сейчас. Не wishlist, не анонсы — только игры с ценником и кнопкой «Add to Cart» в сторе.

Количество: **от 4 до 8 карточек**. Если доступных предзаказов меньше 4 — добиваем горячими новинками первой недели продаж (только что вышедшие хиты).

### Пересчёт в рубли

Для каждого региона своя формула:

**Турция:**
```
Цена RUB = (Цена TRY × курс CBR TRY/RUB × коэффициент наценки) + 250₽
```
Коэффициент наценки — зависит от типа продукта (для игр ориентируемся на ×1.4–1.5 над курсом CBR, по аналогии с подписками).

**Украина:**
```
Цена RUB = (Цена UAH × курс CBR UAH/RUB × коэффициент наценки) + 250₽
```

**Индия:**
```
Цена RUB = (Цена INR × курс CBR INR/RUB × коэффициент наценки) + 250₽
```

250₽ — комиссия менеджера, уже заложена в конечную цену.

Курсы берём с CBR API: `https://www.cbr-xml-daily.ru/daily_json.js` — обновляется ежедневно.

## 3. Расписание обновлений

### Плановые проверки — каждую неделю:
**1, 8, 15, 22, 29 числа каждого месяца**

Агент проверяет все источники (см. раздел 5) и обновляет список:
- Убирает игры, которые уже вышли (дата релиза прошла)
- Добавляет новые предзаказы, у которых появилась цена
- Обновляет цены (Sony иногда меняет цены до релиза)
- Пересчитывает рубли по свежему курсу CBR

### Внеплановые обновления — по событиям:

Триггеры для немедленного обновления (в течение 1–2 часов):
- **State of Play** (Sony) — обычно февраль, май, сентябрь
- **Game Awards** — декабрь
- **Старт предзаказа хайповой игры** (GTA 6, Marvel's Wolverine и т.д.)
- **Публикация третьего трейлера GTA 6** — после него ожидается открытие предзаказа

При срабатывании триггера агент:
1. Проверяет PS Store TR/UA/IN на появление цены
2. Если цена есть → добавляет карточку на главную
3. Публикует новость в новостном блоке сайта
4. Постит в Telegram-канал @PS_PLUS_RUS и VK /activeplay

## 4. Правила отбора и сортировки

### Какие игры попадают в блок:
1. Есть на PS Store Turkey с реальной ценой (не wishlist)
2. Дата выхода — в пределах ближайших 6 месяцев
3. AAA или крупные AA-тайтлы (не инди за ₺200)

### Порядок сортировки:
1. Ближайшая дата выхода — первая
2. При равной дате — по «хайповости» (больше поисковых запросов = выше)
3. PS5-эксклюзивы приоритетнее мультиплатформы

### Когда убираем:
- Игра вышла → убираем из предзаказов через 3 дня после релиза
- Игра перенесена на неопределённый срок → убираем
- Цена снята со стора → убираем

### Особые случаи:
- **DLC/расширения** (типа Diablo IV: Lord of Hatred) — включаем, только если это крупное DLC уровня полноценной игры
- **Ремастеры/коллекции** (типа MGS Master Collection) — включаем
- **Переиздания с PS4 на PS5** — не включаем

## 5. Источники данных (приоритет сверху вниз)

### Основной источник цен:

**PlatPrices API** (бесплатно, 500 запросов/час)
- Запросить ключ: contact@platprices.com
- Регионы: TR (`tr-tr`), UA (`ru-ua`), IN (`en-in`)
- Даёт: цену, наличие скидки, историю цен, обложку, ссылку на стор
- URL: `platprices.com/api.php?key=KEY&ppid=GAME_ID&region=tr-tr`

**Резервный:** PSprices B2B API ($100/мес)
- 71 регион, REST API, ежедневное обновление
- URL: `psprices.com/api/b2b/`
- Подключать, если PlatPrices не хватает или нужна история

### Источник списка предзаказов (какие игры появились):

**PS Store — раздел предзаказов по регионам:**
- Турция: `store.playstation.com/en-tr/category/pre-orders`
- Украина: `store.playstation.com/ru-ua/category/pre-orders`
- Индия: `store.playstation.com/en-in/category/pre-orders`
- США (эталон): `store.playstation.com/en-us/category/pre-orders`

⚠️ PS Store — это React SPA, напрямую парсить сложно (403 для ботов). Варианты:
1. Headless browser (Puppeteer/Playwright) — работает, но тяжело
2. API PlatPrices/PSprices — они уже парсят за нас, берём готовое
3. Ручная проверка (на первом этапе, пока нет агента)

### Источники новостей и триггеров:

| Источник | URL | Что мониторим |
|----------|-----|---------------|
| PlayStation Blog | blog.playstation.com | State of Play, анонсы предзаказов |
| PushSquare | pushsquare.com | Новости PS5, даты релизов, RSS |
| VGC (VideoGamesChronicle) | videogameschronicle.com/guide/new-game-release-dates/ | Сводная таблица дат |
| GamesRadar | gamesradar.com/upcoming-ps5-games/ | Список предстоящих PS5 игр |
| PlayStation Store напрямую | store.playstation.com/en-tr/ps5/games/#pre-order | Визуальная проверка (ручная) |

### Источник курсов валют:

**CBR API:** `https://www.cbr-xml-daily.ru/daily_json.js`
- Обновляется ежедневно
- Содержит TRY/RUB, UAH/RUB, INR/RUB
- JSON, легко парсить
